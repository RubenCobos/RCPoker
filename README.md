RCPoker
=======

Project
-------

A text-based Poker game.

Platform
--------

Cross-platform (tested on Windows 11).

Java SDK 21.0.2

Built on Netbeans 25 with Ant.

Description
-----------

Follow the prompts to play a game of Poker.

Usage
-----

Open a command prompt and navigate to RCPoker.jar.

Type the following:

> java -jar RCPoker.jar
